Context Stats
====

**Context Stats**[^1] lets you display lists of tracks, albums or artists *sorted by statistics* from the context menus of artists, albums, genres, years and playlists.<br><br>For more general library statistics (e.g. genres with the most or best rated tracks), have a look at the [**Visual Statistics**](https://github.com/AF-1/#-visual-statistics) plugin.
<br><br>
## Requirements

- LMS version >= 8.**4**
- LMS database = **SQLite**

## Installation

### Using the repository URL

- Add the repository URL below at the bottom of *LMS* > *Settings* > *Plugins* and click *Apply*:<br>
[https://raw.githubusercontent.com/AF-1/sobras/main/repos/lmsghonly/public.xml](https://raw.githubusercontent.com/AF-1/sobras/main/repos/lmsghonly/public.xml)

- Install the plugin from the added repository at the bottom of the page.
